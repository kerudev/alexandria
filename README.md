# 🏛️ alexandria

Alexandria is my personal library of knowledge where I store and curate info
that is relevant for me, such as situational Linux commands or research about
complex topics.

## Motivation

I've lost count of how many times I needed to use a certain Linux command or
read about a certain topic, only to find myself looking for the source that I
consulted originally.

Sometimes I never find it again. That's why I've decided to write my own
knowledge library and expand it overtime, adding more info or links to trusted
or genuinely good sources.

## About curation and AI

The contents of this library are written and curated by me. None of the files
here are generated by AI.

For me AI, is helpful to get basic knowledge faster using a search engine,
where sometimes the answer I need is buried below lots of search results.

Other times, I just find it hard to find or explain what I'm looking in roughly
10 words that optimize search results.

I prefer to use AI as a starting point and still do my own research about the
topic I want to learn about. I LOVE to dive deep into official sources to
consolidate my knowledge, and that is what I will write about here.
